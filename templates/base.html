About These Rankings

Base Score Components:

  • Points Percentage (Weight: 40%)
  • Goal Differential (Weight: 25%)
  • Special Teams Performance (Weight: 20%)
  • Recent Performance (Weight: 15%)

Bonus Factors:

  • Road Wins (+0.5 per win)
  • Comeback Victories (+0.3 per win)
  • One-Goal Game Success (+0.2 per win)

Data Update Frequency:

Rankings are automatically updated every 30 minutes using the NHL's official API data.
